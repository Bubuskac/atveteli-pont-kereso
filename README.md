# Átvételi pont térképes választó (React + TS)

## Tech stack

- React + TypeScript (Vite)
- Apollo Client – GraphQL kommunikáció
- react-leaflet + Leaflet – térkép
- Nominatim – egyszerű geokódolás
- (Opcionális) Marker clustering csomag nagy elemszámhoz

## Futtatás

```bash
npm install
npm run dev
